👥 Collaborateurs concernés—
L'export vers Silae / Sage / ADP consolide toutes les semaines validées du mois (pas seulement la semaine affichée).
ℹ️ Comment fonctionne l'intégration avec Silae ?
1
Connexion API Silae
Pulsiia appelle l'API REST Silae (/api/v1/variables-paie). La clé API se génère dans Silae → Paramètres → Accès API, puis se colle dans Pulsiia → Paramètres → Intégrations.
2
Mapping des rubriques
Chaque type de variable Pulsiia est traduit en code rubrique Silae : HS25, NUIT20, ABS_MALADIE… Configurable dans Paramètres → Intégrations → Mapping.
3
Traçabilité & erreurs
Seules les variables Validé sont envoyées. Chaque export génère un log horodaté. En cas de rejet Silae, une alerte rouge apparaît dans Pulsiia avec le détail de l'erreur.
💡 Sage / ADP : génère un fichier CSV au format matricule;rubrique;valeur;période à importer directement dans votre logiciel.
